================================================================================
                    BACKEND UPLOADS DIRECTORY
================================================================================

This folder contains all uploaded images served by the backend.

STRUCTURE:
├── clubs/              → Club-specific uploaded images
│   ├── ieee-logo.png
│   ├── ieee-cover.jpg
│   ├── gdg-logo.png
│   └── gdg-cover.png
│
└── defaults/           → Default placeholder images
    ├── logo-default.png     (NEEDED - create a 200x200 placeholder)
    ├── cover-default.jpg    (NEEDED - create a 1920x600 placeholder)
    ├── about-default.jpg
    ├── history-default.jpg
    ├── mission-default.jpg
    ├── activities-default.jpg
    ├── achievements-default.jpg
    └── join-default.jpg

NOTE: You need to add logo-default.png and cover-default.jpg to defaults/
